RFDoc [1] is a web based system for storing and searching Robot Framework [2] 
test library and resource file documentations. It is implemented using Django
web framework [3].

RFDoc code is provided under Apache License 2.0 [4] and its documentation and
tests under Creative Commons Attribution 3.0 license [5].
